# Webpage_project.html-css
# Personal Portfolio — Tatum Booysen

A simple, responsive personal portfolio built with HTML, CSS, and a bit of JavaScript. It highlights my skills, background, and contact information as a practicing software developer from Cape Town, South Africa.

## Live Demo

View the site here: https://tatumbooysen7-commits.github.io/Webpage_project.html-css/

## Features

- Hero section with introduction and call-to-action links
- Skills section with technologies I’m learning and currently using
- About section with a short bio and key highlights
- Contact section with email, location, and social links
- Smooth reveal animations using Intersection Observer

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- Font Awesome for icons
- Google Fonts (DM Sans, Playfair Display)

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/tatumbooysen/webpage_project.html-css.git
   ```
2. Open `index.html` in your browser.
